# DropdownMenu

Dropdown menu component

#### Example

<Canvas of={DropdownMenu.Primary} />

## Usage

```ts
import {DropdownMenu} from "@alextheman/components";

function Example() {
  return (
      <DropdownMenu>
        <MenuItem>Item 1</MenuItem>
        <MenuItem>Item 2</MenuItem>
      </DropdownMenu>
  );
};

export default Example;
```

#### Arguments

- **children** - A list of menu items to display in the dropdown menu
